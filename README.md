# 🌍 Travel & Tour Website (Moliva Inspired)

A modern and responsive travel and tour booking website built with **React + TypeScript + TailwindCSS**.
This project replicates a premium travel UI with a clean design, interactive components, and scalable structure.

---

## ✨ Features

* 🎯 Modern Hero Section with search & category dropdown
* 🖼️ Image carousel for featured destinations
* 📊 Dynamic stats section
* 📂 Category-based search filtering UI
* 🧭 Responsive Navbar with dropdown & mega menu
* 🦶 Pixel-perfect footer (Moliva-inspired)
* ⚡ Fast and optimized UI using TailwindCSS
* 🔐 TypeScript-safe components

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** TailwindCSS
* **Icons:** Lucide React
* **Build Tool:** Vite

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Travel-And-tour.git
cd Travel-And-tour
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm run dev
```
